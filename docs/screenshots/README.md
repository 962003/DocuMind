**Tips for recruiter-friendly screenshots**
- Use a clean browser window (no bookmarks bar, no dev tools).
- Capture at a consistent width (~1280px) so the README grid looks uniform.
- Show real content — a recognizable PDF and a meaningful answer with citations.
